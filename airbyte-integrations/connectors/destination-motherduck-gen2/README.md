# MotherDuck Gen2 Destination

This is the repository for the MotherDuck Gen2 destination connector, written in Kotlin using the Airbyte Bulk CDK.

## Local Development

### Prerequisites
- Java 21
- Gradle

### Building
```bash
gw build
```

### Running Tests
```bash
gw test
```

### Running Integration Tests
```bash
gw integrationTest
```
